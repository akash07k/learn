# Glossary

The canonical definitions for this subject. Add a term the first time a guide uses it as if defined,
in alphabetical order, and link it on first use. Keep the wording identical across the corpus so the
reader builds one mental model.

## Guide

A numbered, self-contained Markdown document under `guides/` that teaches one area in reading order.
The backbone of the subject.

## Operator

The person following the guides. See `CONTEXT.md` for the full role definition.
