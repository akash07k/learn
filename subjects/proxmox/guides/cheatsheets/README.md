# Cheatsheets: fast command lookup

These are terse reference cards for after you have already learned the concept in the guides. Each
card groups a tool's everyday commands by task and points back to the guide that actually teaches
it, so this is the place to look up the exact command quickly, not the place to learn it. The cards
are reference material, not a reading order: jump straight to the tool you want.

Every command here is plain text you run over SSH. None of it needs the web GUI.

- [qm](qm.md) -- create, run, snapshot, clone, and reach virtual machines.
- [pct](pct.md) -- create, run, snapshot, and get inside LXC containers.
- [pvesh](pvesh.md) -- drive the whole Proxmox REST API from the shell.
- [pvesm](pvesm.md) -- inspect and manage storage.
- [pbs](pbs.md) -- the backup client and the Backup Server manager.
- [pveum](pveum.md) -- users, API tokens, roles, and ACLs.
- [networking-firewall](networking-firewall.md) -- interfaces, reloads, and the host firewall.
- [btrfs](btrfs.md) -- the host root filesystem: inspect, scrub, snapshot.
- [zfs](zfs.md) -- the optional secondary pool: pools, datasets, send and receive.
